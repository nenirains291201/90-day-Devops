6. /tmp

Purpose:

Stores temporary files created by applications and users.
Contents may be deleted automatically after reboot.

Example Contents:

ls -l /tmp

Example output:

tempfile1
cache-data

I would use this when:

Working with temporary files during testing and troubleshooting.

7. /bin
Purpose:
Contains essential command binaries required for basic system operation.

Example Contents:

ls -l /bin

Example output:

ls
cp
mv
cat

I would use this when:
Running fundamental Linux commands.


8. /usr/bin
Purpose:
Contains most user-level command binaries and applications.

Example Contents:

ls -l /usr/bin

Example output:

vim
python3
git
curl

I would use this when:
Running installed software and development tools.


9. /opt
Purpose:
Stores optional or third-party software packages.

Example Contents:

ls -l /opt

Example output:

google
custom-app

I would use this when:

Installing or managing third-party applications.
